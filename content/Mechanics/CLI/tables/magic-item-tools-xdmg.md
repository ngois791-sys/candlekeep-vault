# Magic Item Tools
*Source: Dungeon Master's Guide (2024) p. 220* 

**Magic Item Tools**

| Item Category | Required Tool |
|---------------|---------------|
| Armor | [Leatherworker's Tools](Mechanics/CLI/items/leatherworkers-tools-xphb.md), [Smith's Tools](Mechanics/CLI/items/smiths-tools-xphb.md), or [Weaver's Tools](Mechanics/CLI/items/weavers-tools-xphb.md) depending on the kind of armor as noted in the tools' descriptions |
| Potion | [Alchemist's Supplies](Mechanics/CLI/items/alchemists-supplies-xphb.md) or [Herbalism Kit](Mechanics/CLI/items/herbalism-kit-xphb.md) |
| Ring | [Jeweler's Tools](Mechanics/CLI/items/jewelers-tools-xphb.md) |
| Rod | [Woodcarver's Tools](Mechanics/CLI/items/woodcarvers-tools-xphb.md) |
| Scroll | [Calligrapher's Supplies](Mechanics/CLI/items/calligraphers-supplies-xphb.md) |
| Staff | [Woodcarver's Tools](Mechanics/CLI/items/woodcarvers-tools-xphb.md) |
| Wand | [Woodcarver's Tools](Mechanics/CLI/items/woodcarvers-tools-xphb.md) |
| Weapon | [Leatherworker's Tools](Mechanics/CLI/items/leatherworkers-tools-xphb.md), [Smith's Tools](Mechanics/CLI/items/smiths-tools-xphb.md), or [Woodcarver's Tools](Mechanics/CLI/items/woodcarvers-tools-xphb.md) depending on the kind of weapon as noted in the tools' descriptions |
| Wondrous Item | [Tinker's Tools](Mechanics/CLI/items/tinkers-tools-xphb.md) or the tool required to make the nonmagical item on which the magic item is based |
